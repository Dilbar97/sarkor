# sarkor

## Локальное развёртывание

```
1) Для начала скопируйте config.local.yaml и вставьте в том же директории и 
   у копии уберите из названия '.local'
2) В скопированном конфиг файле вставьте свои значения
```

## Запуск

```
1) go mod tidy
2) Здесь по хорошему должен быть swag init, но тут я решила все swag файлы 
   залить в репу. Хотя так делать не очень удобно в реальных крупных проектах
3) go run main.go. Здесь будет: 
   - Выстраиваться роутинг
   - Формироваться свагер
   - Поднимаются миграции
```

## Сваггер

```
В браузере можно открыть следующую ссылку 
http://{server_host}:{server_port}/swagger/index.html
```
